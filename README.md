# Analyzing-Data-with-Modules-and-Syntax-JS-Lab
## Overview
This lab demonstrates the use of JavaScript modules, spread operators, and array manipulation to merge multiple arrays of usernames into a single array. The solution returns an object containing the merged user list and the date of merging.

## Problem Statement
A social media company has merged with another company and needs to combine all user accounts into one master list. The function should:
- Accept an undetermined number of arrays
- Merge all arrays into one
- Include today's date
- Return the result as an object

- Node.js
- npm (for testing)
